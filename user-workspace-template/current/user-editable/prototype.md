## Structured Screen Content

Use these supported element types when generating the prototype page JSON:

- `text`: eyebrow, title, body, tone.
- `field`: label, value or placeholder, optional meta or status.
- `list`: title and items; items may include title, body, meta, or status.
- `card`: eyebrow, title, body, meta, tags, tone.
- `notice`: title, body, tone.
- `actions`: primary and optional secondary action labels.
- `progress`: title, numeric value from 0 to 100, optional display value.

Supported tones are `neutral`, `accent`, `success`, `warning`, and `danger`.
