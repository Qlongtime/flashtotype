# User-Editable Workspace

Edit files in this folder during Flashtotype work.

## Core Files

- `flashtotype-brief.md`: product thesis and PM/PO context.
- `decision-pack.md`: concise team-ready recommendation.
- `evidence.json`: source and claim ledger.
- `user-journey.md`: source for the board journey page.
- `prototype.md`: source for the board prototype page.
- `Design.md`: source for the board design system page.
- `presentation.md`: source for the board presentation page.
- `flashtotype-library.md`: selected frameworks and suggestions.

## Input Folders

- `references/`: research notes, interview notes, links, copied source summaries.
- `data/`: CSV, JSON, exports, metrics tables, survey results.
- `assets/`: screenshots, diagrams, brand images, design references.

Generated presentation images are output files, not source files. They should be saved under `../output/assets/` during a real run.

Do not edit `.flashtotype/` during normal work. Open `../output/index.html` to view the board, and ask the agent to regenerate output from this folder after source changes.
